## SIGNATURE: Ghost text (280px opacity 0.04) on every slide + diagonal color band (-1.5deg skew)
## AMBIENT: Floating geometric shapes (circles/triangles border-only, opacity 0.10) + star sparkles (✦)
## LAYOUT IDENTITY
```
FORBIDDEN: uniform grids, same-size text blocks, centered safe layouts
MANDATORY: ghost text (280px, opacity 0.04) EVERY slide, diagonal color band (skew -1.5deg),
  bold condensed Bebas Neue, asymmetric layouts, at least 1 oversized element bleeding off edge
SLIDE IDENTITY:
  Cover: massive condensed type + diagonal orange band + ghost text bg
  Content: asymmetric split + oversized element bleeding off edge
  Chapter: single massive word filling 80% height + ghost number
  Data: giant numbers + small labels + diagonal band accent
FONT: Bebas Neue 400 (display — it's naturally bold/condensed), Nunito 400/800 (body)
```
