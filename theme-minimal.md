## LAYOUT IDENTITY
```

SLIDE IDENTITY:

  Content: clean 2-col text, no cards, gold hairline separator
  Chapter: single elegant word + 80px gold rule beneath

FONT: Cormorant Garamond 600 ONLY (never 900 — restraint is the point), Jost 300/400 body
```
